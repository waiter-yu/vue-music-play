## 安装
```
$ npm install vue-music-play
```
## 使用
```
 <music-play musicUrl=""/>
```
 ```
import MusicPlay from 'vue-music-play'
import 'vue-music-play/dist/vue-music-play.css'
components: {
    MusicPlay
  }
 ```
#### 参数说明
```
musicUrl  :音乐url
```
#### 使用之前请先注入ElementUI
